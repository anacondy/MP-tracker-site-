## â“ Troubleshooting

### "Install" option not appearing?
- Make sure you're using a supported browser (Chrome, Edge, Safari)
- Check that you're on HTTPS (the site must be secure)
- Try clearing browser cache and reloading

### App not updating?
- Force refresh: Ctrl+Shift+R (Windows/Linux) or Cmd+Shift+R (Mac)
- Clear the app's cache through browser settings
- Uninstall and reinstall the app

### Offline mode not working?
- Visit all pages while online first (they need to be cached)
- Check that Service Worker is registered in browser DevTools
- Some features require internet (live data updates)
